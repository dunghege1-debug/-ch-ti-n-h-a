# Tadpoles Tales

Game Python viết bằng Pygame.

## Cách chạy
1. Cài Python 3.10
2. Cài thư viện:
   pip install pygame mediapipe opencv-python
3. Chạy file:
   python game của Dũng mới.py

## Điều khiển
- WASD: di chuyển ( chọn 3 cấp độ)
- Finger Control: điều khiển bằng ngón tay (camera)(khi chọn nút trên cùng bên trái)
